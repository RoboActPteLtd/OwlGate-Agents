# owlgate-gate (UiPath coded agent)

The deployable **UiPath coded agent** form of OwlGate — a Coded Function that wraps
`OwlGatePipeline` and returns the go / no-go verdict. Published to UiPath via the
`uip` CLI; the brain itself is the deterministic pipeline from the parent package.

## Layout

- `main.py` — the `main(GateIn) -> GateOut` entrypoint (dataclass IO).
- `owlgate_agents/` — **vendored** copy of the parent package (`../src/owlgate_agents`),
  so the agent packs self-contained (the package isn't published to PyPI). Refresh
  with: `Copy-Item -Recurse -Force ../src/owlgate_agents ./owlgate_agents` then delete
  `owlgate_agents/__main__.py`.
- `catalogues/` — bundled default test catalogue.
- `uipath.json` / `entry-points.json` / `bindings.json` — UiPath project config
  (generated by `uip codedagent init`).
- `evaluations/` — smoke eval set + a deterministic JSON-similarity evaluator.

## Develop / run / deploy

```bash
uv venv --python 3.13 && . .venv/Scripts/activate   # (bash: source .venv/bin/activate)
uv sync
uip codedagent init                                  # regenerate schemas after IO changes
uip codedagent run main -f examples/run-input.json --output-file out.json
uip codedagent eval main evaluations/eval-sets/smoke-test.json --no-report
uip codedagent deploy --my-workspace                 # publish (bump version to re-deploy)
```

Input (`GateIn`): `diff: [{path, lines}]`, optional `catalogue: [suite...]`
(defaults to the bundled one), optional `outcomes: {suite -> "pass" | {message,...}}`.
Output (`GateOut`): `verdict`, `needs_human`, and the full `report`.
